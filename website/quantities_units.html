üìè Working with Quantities & Units

The application uses a set of predefined default units for each quantity type (e.g., length, pressure, temperature).

üîÅ Input Flexibility

When using the user interface, you can input values using different units. The software will automatically convert your input into the default unit for internal calculations.

üíª Scripting with Lua

When writing Lua scripts, always provide values using default units. If you want to use a different unit in your code, you must manually convert it using the command:

ConvertUnit(value, fromUnit, toUnit)

‚ö†Ô∏è Default units are always the first unit defined for each quantity in the internal database.

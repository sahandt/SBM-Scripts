Liquid Distribution Systems

The Liquid Distribution System module allows you to design and simulate piping systems that carry fluids throughout a facility. These systems can include appliances, tanks, pumps, valves, and pipes connected via nodes and joints. Each element can be customized, and the entire system can be exported for hydraulic simulation using EPANET.

EPANET Conversion

Elements in the liquid distribution system can be converted to an EPANET input file, which is compatible with the EPANET software for detailed hydraulic analysis. You may also perform simulations directly using EPANET Toolbox functions through Lua scripting. These Lua functions follow the structure of the EPANET API from EPANET official site, but return the output value instead of an error code. Please see the API guide and the EPANET converter example on the GitHub repository for more information.

Menu Guide

  • Pipe Menu
    • Add Pipe — Place a standard pipe to carry liquids through the system.
    • Add Insulated Pipe — Add a pipe with thermal insulation to reduce heat loss or gain.
    • Add Pipe Joint — Insert a connection point to join multiple pipes.
  • Valve Menu
    • Check Valve — Prevent reverse flow to ensure liquid only travels in one direction.
    • Flow Control Valve — Regulate the flow rate of liquid through the system.
    • Pressure Breaker Valve — Reduce pressure surges to protect downstream components.
    • Pressure Reducing Valve — Lower high inlet pressure to a safer, steady output level.
    • Pressure Sustaining Valve — Maintain a minimum upstream pressure by restricting downstream flow.
    • Throttle Control Valve — Precisely adjust liquid flow using partial closure of the valve.
    • General Purpose Valve — Use for basic on/off or manual control of liquid flow.
  • Tank Menu
    • Rectangular Tank — Add a tank with a box-like shape for storing or processing liquid.
    • Vertical Cylindrical Tank — Place a standing cylindrical tank ideal for space-saving storage.
    • Horizontal Cylindrical Tank — Insert a horizontal tank commonly used for transport or stability.
  • Appliance Menu
    • Hydraulic Appliance — Add a generic device that interacts with or consumes liquid, such as heaters or coolers.
    • Pump — Insert a pump to move liquid through the system by increasing pressure or flow.

Structural Systems

The structural system in the Sustainable Building Modeller allows you to model buildings, walls, structural links, and nodes for analysis and design purposes. It also supports dynamic physical elements that can interact with their environment through physics simulation.

Dynamic Elements

Dynamic elements are customizable elements that can move freely, simulate physical behavior, or be bound by constraints. They can be assigned mass, velocity, and other physical properties, making them useful for simulating objects that interact dynamically with their surroundings.

Adding Building Elements

  • When you add a new building, it becomes the active network. Be sure to select the right network for additional elements.
  • The first created building automatically defines these networks: Column-Floor, Floor-Column, and Room-Column.
  • Each building includes a default floor at the selected elevation. You can add additional floors with custom heights and properties.
  • To add columns, select points and assign their base and top elevations or floors.
  • Create walls by selecting two columns and two elevation levels. The direction of wall layers depends on the order of column selection.
  • Insert doors, windows, and wall openings using the Openings menu with a selected wall.

Structural Links and Nodes

Structural links and nodes are used for frame analysis, independent of building components. Use them to define structural frames compatible with analysis tools like Frame3DD.

See the Frame3DD converter script on GitHub for export options.

Furniture, Fixtures, and Vehicles

Model interior and exterior objects using basic shapes like cubes, cylinders, and spheres. These can be combined using the Attach function to build more complex forms.

Vehicles are special elements that can be controlled by the user. Right-click a vehicle and choose Drive to operate it. Press Escape or click outside the vehicle to stop. Note that pressing Escape also resets the vehicle's position.

Structural System Menu Guide

Building Menu

  • Add Building — Create a new building element.
  • Add Floor — Add a floor to the selected building.
  • Add Square Column — Add a basic square column.
  • Add Column — Add a general-purpose column.
  • Add Profiled Column — Use a predefined profile shape for a column.
  • Add Wall — Add a wall between columns and elevation levels.

Opening Menu

  • Add Door — Insert a door into a wall.
  • Add Window — Insert a window into a wall.
  • Add Rectangular Wall Opening — Create a custom opening in a wall.

Zones Menu

  • Add Room — Define a room in the model.
  • Add Zone — Group multiple rooms into a zone.

Custom Menu

  • Add Furniture — Add generic furniture to the layout.
  • Add Fixture — Add and combine elements to model fixtures.
  • Add Dynamic Element — Insert a physical object with mass, motion, and constraints.
  • Add Vehicle — Add a controllable vehicle model.

Structural Network Menu

  • Add Structural Node — Insert a node for frame analysis.
  • Add Structural Link — Connect two nodes using a beam link type.

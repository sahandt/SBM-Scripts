General App Menus

This page provides an overview of the main application menus in the Sustainable Building Modeller. Each menu gives access to tools for creating, aligning, scripting, managing, and navigating within the project environment.

1. Secondary Commands (Bottom Bar)

These are global commands accessible at any time.

  • Create New Project:
    • New Metric Project: Start a new project using the metric system.
    • New Imperial Project: Start a new project using the imperial system.
  • Project File Operations:
    • Open Project: Load an existing project from storage.
    • Save Project: Save current project changes.
    • Save As Project: Save the project under a new name or location.
    • Open Backup: Restore from a backup version of the project.
  • Settings: Open the application configuration panel.
  • Buy Add-ons: Purchase optional modules or remove ads.
  • Help Menu:
    • Keyboard Controls: View shortcut keys and interaction methods.
    • Visit Blog: Open the official blog in a browser window.
    • External Links: Access useful online resources.
    • Important Notes: Display warnings or relevant usage advice.
    • User Agreement: Show the software license and usage policy.
    • Component Notices: Acknowledge external libraries or components used.
    • About the App: Show version, credits, and application information.

2. System Selection

The System Selection menu is used to switch the user interface to work on a specific system such as HVAC, Electrical, or Structural. This affects which menus, tools, and options are shown in the app interface.

Elements are automatically assigned to the appropriate system based on their type, and this assignment cannot be changed manually by the user.

3. Manage Menu

Contains tools for project-level management and access to internal data libraries.

  • Project Management:
    • BOM (Bill of Materials): Generate a materials list based on the project model.
    • Cost Estimation: Estimate material or component costs.
    • Project Timeline: Manage project milestones and schedules.
  • Data Libraries:
    • Product Database: Access a catalog of standard components.
    • Material Library: View and assign material types and physical properties.
    • AISC Shapes: Insert predefined structural steel profiles.
  • Log Management: Inspect logs of activity, warnings, or system output.

4. Script Menu

Provides tools to automate tasks and simulate behaviors using Lua scripts.

  • Run Lua Script: Execute a Lua script file.
  • New Script: Create a new Lua script from scratch.
  • Edit Script: Open and modify an existing script.
  • Run Project Script: Run scripts that are saved as part of the current project.
  • Custom Calculator: Open a user-defined calculator tool based on Lua.
  • Load Lua Script:
    • DXF Converter: Converts the current model to an AutoCAD DXF file.
    • EPANET Converter: Converts water network models into the EPANET .inp format.
    • Modelica Fluid Converter: Converts the current model to Modelica models.
    • Frame3DD Converter: Converts the structural elements into a Frame3DD input file.

5. Align Menu

Contains tools for geometric alignment, movement, and measuring between model elements.

  • Align with Project Coordinate System:
    • Align X: Align selected elements along global X-axis.
    • Align Y: Align elements along global Y-axis.
    • Align Z: Align elements along global Z-axis.
  • Align with Element Coordinate System:
    • Align Local X: Align to the local X-axis of a reference element.
    • Align Local Y: Align to the local Y-axis of an element.
    • Align Local Z: Align using local Z direction.
  • Element Movement Tools:
    • Move Element: Move the selected object's origin to a selected point.
    • Move Element Horizontally: Move the selected object's origin to a selected point, but it will remain in its XY plane.
    • Attach Element: Attach an object to another by face or point.
  • Measurement Tools:
    • Measure Point: Capture coordinates of a single point.
    • Measure Line: Measure distance or angles between two points.

6. View Menu

Switch between standard perspectives and define working planes for modeling operations.

  • Map View: Switch to the satellite or terrain-based map background.
  • Orthographic Views:
    • Top View: Switch to a plan view looking down.
    • Bottom View: View from beneath the model.
    • Left View: Side view from the left.
    • Right View: Side view from the right.
    • Front View: View the model from the front.
    • Back View: View the model from behind.
  • Set Working Plane:
    • Tangent to Element: Set modeling plane parallel to a selected face.
    • Element XY: Set the plane to the XY plane of an element.
    • Element YZ: Use the element’s YZ plane for new drawing.
    • Element XZ: Align working plane to the element’s XZ surface.
